음성 샘플 평가 안내

안녕하세요. 음성 샘플 평가에 참여하신 분들께 감사 인사드립니다.

참여자 분들은 5개 파트, 640문항으로 이루어진 음성 평가를 진행하시게 됩니다.

예상 소요 시간은 총 2시간 가량으로, 각 파트별로 아래의 링크에 접속하여 평가를 진행하시기 바랍니다.

평가를 진행하기 전, 각 파트 페이지 위의 안내문을 반드시 숙지하고 평가를 진행하시기 바랍니다.

평가 페이지를 새로고침할 경우 평가한 내역이 사라지니 주의하시기 바라며, 문의가 있다면 해당 안내를 전달한 담당자에게 연락 부탁드립니다.

파트 1: 음질 테스트(120문항)

파트 2: 음질 테스트(120문항)

파트 3: 화자 유사도 테스트(120문항)

파트 4: 화자 유사도 테스트(120문항)

파트 5: 감정 유사도 테스트(160문항)

평가에 참여해주신 분들께 다시 한 번 감사의 말씀 드립니다.
